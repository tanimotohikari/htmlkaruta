## 今回の肝　Socket.io
### Socket.ioとは？
リアルタイムなWebアプリを実現するためのnode.js用サーバー側ライブラリとブラウザ用JavaScriptライブラリのセットです。
リアルタイムに反映されるチャットなどがイメージしやすいと思います。
